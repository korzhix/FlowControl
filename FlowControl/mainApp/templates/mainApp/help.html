О приложении

Приложение позволяет эффективнее организовывать учебный процесс, разбивая его наа потоки и точечно контролируя каждый из них, а контроль всех потоков и есть контроль учебного процесса. Для этого в приложении собрано несколько инструментов: сервис хранилища данных, сервис заметок с возможностью создания ярлыков для каждого предмета и удобной ссылкой на них из бокового меню, сервис календаря, для отметки важных событий, и удобный способ мониторинга брс.
Приложение ориентировано на студентов ЮФУ, имеет синхронизацию с БРС, расписанием и локальным хранилищем данных института. Каждый поток – это предмет из вашего расписания. После регистрации на сайте заполните учетные данные для входа в личный кабинет системы БРС, чтобы наше приложение смогло синхронизироваться с системой и получить данные. Для этого в личном кабинете нажмите на кнопку “Ввести учетные данные openid” и заполните соответствующие поля.
Доступна возможность самостоятельно выбирать каждый из трех сервисов в разделе настроек, там же можно задавать и настройки каждого потока.

Настройки

Общие настройки

В общих настройках вы можете настроить сервисы, которые будут использованы по умолчанию.
Сервис заметок: введите ссылку на желаемый сервис заметок.
Сервис хранилища: введите ссылку на желаемый сервис хранения данных.
Календарь: аналогично.
*P.S. К сожалению не все сервисы способны работать в нашем приложении, это обусловленно техническими трудностями. Но не расстраивайтесь и попробуйте другой вариант.

Настройки потока

В настройках потока можно настроить каждый поток, для этого в разделе Настройки→Настройки потока в выпадающем списке выберите нужный поток и заполните следующие поля:
Домашняя работа – ссылка на ярлык с заметками о домашней работе по выбранному предмету.
Цели – ссылка на ярлые с заметками о целях для данной дисциплины.
To-do – ссылка на список задач по текущему предмету.

Как эффективно организовать учебу

В качестве сервиса заметок по умолчанию приложение использует Nimbus Note (хотя вы можете использовать другой, переопределив сервис в настройках). С этим сервисом вы можете эффективно организовать учебный процесс за несколько простых шагов:

1) Зарегистрируйтесь в Nimbus Note (позже планируется добавить автоматическую регистрацию)

2) Создайте папку для общих заметок, в нее будут помещаться заметки из ненастроенных потоков . Поместите ссылку на эту папку в раздел Сревис заметок на странице настроек.

3) Создайте папку с названием предмета, для которого вы настраиваете поток. В ней также создайте три папки: Домашнее задание, Цели, To-do и поместите ссылки на них в соответствующие разделы настроек потока.

Поток готов, теперь вы можете обращаться к настроенным разделам потока из бокового меню.
